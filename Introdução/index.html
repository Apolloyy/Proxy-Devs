O que é a Proxy Devs?

A Proxy Devs é a unificação entre os dois cursos (jodi e des).
A palavra "Proxy" surge a partir das matérias técnicas que aprendemos
ao longo dos anos letivos, já a palavra "Devs" vêm da palavra desenvolvimento,
assim possuindo uma conexão entre os dois cursos, pois ambos remetem a programação.
